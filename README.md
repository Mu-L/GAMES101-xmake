# GAMES101作业 xmake版本

# 快速开始

自行安装自己平台下的C++编译工具链

安装
[xmake](https://xmake.io/#/zh-cn/guide/installation)

下载代码
```bash
git clone https://github.com/star-hengxing/GAMES101-xmake.git
```

然后可以进入作业目录测试，比如
```
cd src/Assignment1
xmake
xmake run
```

第一次使用会安装第三方库，根据命令行提示操作即可

# 注意事项

- 作业文档在doc目录下
- 程序运行目录均在自己作业目录
- 模型目录放在项目根目录
